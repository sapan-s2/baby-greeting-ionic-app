# baby-greeting-ionic-app




ios package instruction

  1.)  clone the app
  
  2.)  execute following command in the application directory 
       npm install .
      
  3.) execute below command for build 
       ionic cordova build ios
       
       note : you might have to install IOS SDK 
         https://wiki.appcelerator.org/display/guides2/Installing+the+iOS+SDK
